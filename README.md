# Project 6
## Studious

Create courses, study groups, flash cards, and chat with your classmates!

## Setup

### On Ubuntu
```bash
./bin/setup
```

### On macOS
```bash
npm install
bundle install
rake db:create && rake db:schema:load
```

## Start the dev server
```bash
./bin/server
```

Then head to [localhost:5100](http://localhost:5100) and get studious
